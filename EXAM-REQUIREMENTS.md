# 📋 Requisitos Completos del Examen - Databricks Certified Generative AI Engineer Associate

## 🎯 Información Básica del Examen

| Aspecto | Detalles |
|---------|----------|
| **Nombre** | Databricks Certified Generative AI Engineer Associate |
| **Preguntas** | 45 preguntas (opción múltiple o selección múltiple) |
| **Tiempo Límite** | 90 minutos |
| **Formato de Entrega** | Online Proctored (Kryterion Secure Browser) |
| **Validez** | 2 años |
| **Recertificación** | Requerida cada 2 años (examen completo) |
| **Requisito Previo** | Ninguno requerido (se recomienda 6 meses de experiencia hands-on) |
| **Calificación Mínima** | ≥ 70% para aprobar |

---

## 🖥️ Requisitos de Hardware

### Espacio en Disco
- **Mínimo**: 50 MB disponibles
- *Nota*: Necesario para descargar e instalar el Secure Browser

### Pantalla y Resolución
- **Mínimo**: 1024 × 768 píxeles
- **Recomendado**: 1366 × 768 o superior
- **Tipo**: Monitor/Pantalla con conexión visible

### Entrada/Salida
- **Webcam**: Requerida (USB si es externa)
- **Micrófono**: Requerido (USB si es externo)
- **Altavoces**: Para escuchar instrucciones del proctor
- **Teclado**: Inglés (QWERTY)

### Restricciones Importantes
- ⚠️ **NO se permiten máquinas virtuales** (VirtualBox, VMware, Hyper-V, etc.)
- ⚠️ **NO se permiten emuladores** (NoxPlayer, BlueStacks, etc.)
- ⚠️ **Se recomienda usar computadora personal**, no de trabajo

---

## 🌐 Requisitos de Conectividad (Internet)

### Velocidad Mínima
- **Descarga**: 1.0 Mbps
- **Carga**: 1.0 Mbps
- **Recomendado**: Descarga ≥ Carga (simétrico ideal)

### Latencia y Estabilidad
- **Ping**: < 200 ms (máximo)
- **Jitter**: < 50 ms (máximo)
- **Estabilidad**: Conexión dedicada (no WiFi público)

### Recomendaciones Prácticas
1. **Antes del examen**: Testea tu conexión en https://www.kryterion.com/systemcheck/
2. **Cierra otras apps**: Paralizar descargas, actualizaciones, streaming
3. **Usa Ethernet**: Si es posible, conectá con cable LAN (no WiFi)
4. **Reinicia el router**: 30 minutos antes del examen
5. **Cierra VPN/Proxy**: Pueden interferir con el Secure Browser

## 🔒 Secure Browser - Respondus

### Qué es
- Navegador especial para exámenes proctored que bloquea acceso a otras apps
- Desarrollado por **Kryterion** (basado en Respondus)
- Previene: Alt+Tab, copiar/pegar, screenshots, búsquedas web

### Instalación
1. Log in en https://webassessor.com/databricks
2. Navegá a "My Assessments"
3. Clickeá "Install Secure Browser"
4. Seguí las instrucciones en pantalla

**Nota**: Puede requerir permisos de administrador

### Requisitos de Software
- **Windows 10+**: IE, Edge, Chrome, o Firefox reciente
- **macOS**: Safari, Chrome, o Firefox reciente
- **Chromebook**: Chrome actual

### Solución de Problemas
Si el navegador no se instala:
- ✅ Deshabilitá antivirus temporalmente
- ✅ Deshabilitá bloqueadores de pop-ups
- ✅ Deshabilita VPN/Proxy
- ✅ Cierra programas: Skype, TeamViewer, VMware, Bootcamp
- ✅ Loguéate como admin del SO
- ✅ Reinicia la computadora

### Actualizar Secure Browser
Si ves "Update required":
1. Abre Webassessor
2. Navigá a My Assessments
3. Clickeá "Install Secure Browser"
4. Seleccioná "Reinstall"
5. Seguí las instrucciones

## 💡 Consejos Finales

### 1. Timing
- Llegá **15 minutos antes** al horario
- El proctor necesita 10-15 min para verificación
- Esto te deja ~75 min para responder 45 preguntas (~1.6 min/pregunta)

### 2. Estrategia de Respuestas
- Lee atentamente cada pregunta (posibles trampas)
- Marcá preguntas difíciles y volvé después
- No dejes preguntas sin responder
- Para selección múltiple: lee todas las opciones

### 3. Si Algo Sale Mal
- **Conexión perdida**: El navegador se reconecta automáticamente
- **Crash del navegador**: Relanzá Secure Browser (tienes 5 min)
- **Proctor problema**: Contactá al soporte de Kryterion (número en pantalla)

### 4. Después del Examen
- Resultados disponibles en **1-2 días hábiles**
- Si pasaste: certificado descargable
- Si no pasaste: puedes reintentar en 2 semanas
